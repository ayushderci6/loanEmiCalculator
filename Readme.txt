						Loan EMI Calculator

This is a code written in React Js to get a webpage which display an loan amount in text box with an slider, an loan duration text box with an slider and an submit button

When user enter loan amount and duration and press submit button then user will get the interset rate and the monthly amount to be paid on right side 

The external library used are :
1)Axios : to get the api 
2)react-rangeslider : to get the slider



Files in the code :

1) app.js : main file contain all code
2) images : contain all images:
3) app.css : some styles
4) component: contain app presentational content like slider
